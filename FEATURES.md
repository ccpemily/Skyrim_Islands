# Skyrim Islands Mod Features

## 基本功能

这是一个空岛（什么是空岛可以参考Minecraft的Sky Island玩法）mod，旨在让玩家拥有一个可以完全自定义的浮空岛殖民地地图，以及配套的资源生成、收集和殖民地大地图移动玩法。

## 功能分块

### 空岛地图

空岛地图的整体设计可以参考Odssey DLC的近地轨道地图。

空岛地图应设计在与Odssey DLC的近地轨道类似的独立地图层中，初始只有一个较小的由空岛核心（8*8不可摧毁的地形结构，表面是草地贴图，其上可以种植作物）和周围的浮空石平台（这是一种与重型桥梁类似的地基）构成的岛屿，后续可以被玩家扩建。

地图的背景部分应与近地轨道类似，但把地面放得更大，加入云彩效果。

“空”地格实际应该是类似深水和近地轨道真空的不可通行、不可建造地块，我暂且称之为云海，允许玩家在其中划定垂钓区。

地图气候应与所在大地图位置对齐，但没有天气变化（在云层上方）、季节变化更不明显，平均温度应当低于所在大地图位置的地面地图平均温度。

关于发电机：空岛上风力发电机和太阳能板的功率将得到提升，且风力发电机可以不间断发电（但仍应该有一定波动）。

### 游戏机制联动

初始任务：游戏开始后直接发送给玩家，参考超凡枢纽任务线，让玩家触发任务后，删除当前的殖民地，然后选择一个位置新建空岛殖民地。

袭击逻辑：初始不会出现袭击。等待一段时间后，开始出现空投袭击，并发送特殊任务，玩家可以通过完成任务来在地图边缘建造一个袭击平台，其他类型的袭击以及友好商队会在这个平台上出现。如果平台没有连通到主岛核心的路径则会直接坠毁。

机械师任务：替代原版的机械师任务，改为一个触发式任务，完成后召唤坠毁的机械师飞船

异象任务：参考Odssey DLC中飞船剧本对倒塌石碑的处理，改为一个触发式任务，必须在建造袭击平台之后才发送给玩家，玩家完成后可以建造一个异象信标，石碑会在信标位置出现。

### 资源获取

初始资源：给予玩家一定量的生存必需资源（钢铁、包装生存食物、医药等，以及玩家所选择剧本的开局资源）以及一架轨道穿梭机。

木材和作物：可以在空岛核心上种植，后续可以建造*温室*建筑种植更多

金属资源：开局附送一个损坏的*超凡链接节点*和*广域扫描仪*，玩家可以通过任务获取修复节点和扫描仪所需的科技和材料。可通过扫描仪获取特殊矿物地点（就像原版的远距离扫描仪那样，但是更快、范围更大），然后派遣穿梭机前往采矿，或使用修复的链接节点自动开采（链接到目标地点后，删除目标地点，并为链接节点新增一个可工作项目，Pawn可在节点处工作以远程采集资源，直到资源耗尽），链接节点可以通过附属模块升级，等级越高，每次链接的资源量越多，单次工作开采的矿物越多。

石材：与金属的获取基本相同。超凡链接节点升到特定等级后，可以从任意资源点中开采特殊的云海精华，用于制造浮空石平台。钓鱼也有概率获得少量云海精华。

动物和其他：理论上可以通过穿梭机获得。不是特别紧急。

### 空岛移动

通过空岛专属科技解锁浮空推进器和空岛控制终端，建造后可以操作空岛进行类似逆重飞船的发射，但只能在空岛地图层中移动而无法降落到其他地块上。

### 一些特殊机制

空岛能量尖塔：设想的一种电力分配建筑，在核心上固定刷出，修复后启用可使所有连接的浮空石平台并入尖塔所在的电网。无论是否修复，可以派遣小人对尖塔进行调查（属于研究工作），会产出特殊的云海测量数据，用于解锁空岛专属的科技（实现参考异象的异象研究点数）。

空岛招募：会随机触发专门的坠毁空投仓任务，坠毁的Pawn玩家可以选择救助后加入殖民地。殖民地人数越多，触发的几率越低。（可能的trick：安装HumanoidAlienRace后，优先生成HAR友好派系的Pawn）
